生徒向けPC利用ガイド

ホーム

初級者向け案内 (Windows 11)

1. 電源の入れ方と切り方

電源を入れる: PC本体の電源ボタンを押します。

電源を切る（シャットダウン）:

  1. 画面下のスタートボタン（Windowsロゴ）をクリックします。
  2. 表示されたメニューの右下にある電源アイコンをクリックします。
  3. 「シャットダウン」を選択します。
power_settings_new

スリープ: 短時間だけPCから離れる場合は、「スリープ」を選ぶと節電でき、すぐに作業を再開できます。

再起動: PCの調子が悪いときや、ソフトウェアのインストール後に「再起動」を選択します。

2. デスクトップとタスクバー

PCにサインインすると表示される最初の画面が「デスクトップ」です。画面下にある横長のバーを「タスクバー」と呼びます。

desktop_windows
  • スタートボタン: アプリケーションの一覧を表示したり、PCの設定を変更したりできます。
  • 検索: PC内のファイルやアプリ、Web上の情報を検索できます。
  • アプリのアイコン: よく使うアプリをピン留めしておくと、ワンクリックで起動できて便利です。

3. Wi-Fiに接続する方法

  1. タスクバーの右側にある、ネットワークアイコン（扇形のアイコン）をクリックします。
  2. 利用可能なWi-Fiネットワークの一覧が表示されます。
  3. 接続したいネットワーク名（SSID）の横にある「接続」ボタンをクリックします。
  4. パスワード（セキュリティキー）を求められたら、入力して「次へ」をクリックします。
wifi

4. ファイルとフォルダーの基本

「エクスプローラー」というアプリを使うと、PC内のファイルやフォルダーを管理できます。タスクバーの黄色いフォルダーのアイコンをクリックして起動します。

folder_open
  • 新しいフォルダーの作成: 何もないところを右クリックし、「新規作成」>「フォルダー」を選択します。
  • 名前の変更: ファイルやフォルダーを右クリックし、「名前の変更」を選択します。
  • コピー＆貼り付け: ファイルを右クリックして「コピー」を選び、貼り付けたい場所で右クリックして「貼り付け」を選択します。

5. Webブラウザの基本的な使い方

インターネットを見るためのアプリが「Webブラウザ」です。学校のPCでは「Microsoft Edge」が標準で使えます。

public
  • タブ: 複数のWebページを同時に開いておける機能です。上の「+」ボタンで新しいタブを開けます。
  • ブックマーク（お気に入り）: よく見るページを登録しておくと、ワンクリックでアクセスできます。アドレスバーの星マーク☆をクリックすると登録できます。
  • 検索: 上のアドレスバーにキーワードを入力してEnterキーを押すと、インターネット上の情報を検索できます。

6. インターネットを安全に使うために

security

インターネットはとても便利ですが、使い方を間違えると危険なこともあります。以下の点を必ず守りましょう。

  • 個人情報を教えない: 自分の名前、住所、電話番号、学校名、パスワードなどを、知らない人や怪しいサイトに絶対に入力しないでください。
  • 知らない人からのメールやメッセージに注意: 添付ファイルやリンクをむやみに開かないようにしましょう。ウイルスに感染する危険があります。
  • 著作権を守る: インターネット上の文章、写真、イラスト、音楽などには著作権があります。許可なく自分の作品として発表したり、勝手に使ったりしてはいけません。
  • 困ったときは相談する: インターネットを使っていて怖い思いをしたり、困ったことがあったりした場合は、すぐに先生や保護者、信頼できる大人に相談してください。

中級者向け案内 (Windows 11)

ショートカットキーを覚えて、作業をスピードアップしましょう。

基本的なショートカットキー

  • Ctrl + C: 選択した項目をコピーする
  • Ctrl + X: 選択した項目を切り取る
  • Ctrl + V: コピーまたは切り取った項目を貼り付ける
  • Ctrl + Z: 操作を元に戻す
  • Ctrl + A: すべての項目を選択する
  • Alt + Tab: 開いているアプリを切り替える
  • Windows ロゴキー + D: デスクトップを表示/非表示にする
  • Windows ロゴキー + L: PCをロックする

新しい便利なショートカットキー (Windows 11)

  • Windows ロゴキー + W: ウィジェットを表示する
  • Windows ロゴキー + A: クイック設定を開く
  • Windows ロゴキー + N: 通知センターを開く
  • Windows ロゴキー + Z: スナップレイアウトを開く
