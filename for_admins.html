システム管理者向けPC利用ガイド

ホーム

初級者向け案内 (Windows 11)

1. 電源の入れ方と切り方

電源を入れる: PC本体の電源ボタンを押します。

電源を切る（シャットダウン）:

  1. 画面下のスタートボタン（Windowsロゴ）をクリックします。
  2. 表示されたメニューの右下にある電源アイコンをクリックします。
  3. 「シャットダウン」を選択します。

[スクリーンショット: Windows 11のスタートメニューと電源オプション]

スリープ: 短時間だけPCから離れる場合は、「スリープ」を選ぶと節電でき、すぐに作業を再開できます。

再起動: PCの調子が悪いときや、ソフトウェアのインストール後に「再起動」を選択します。

2. デスクトップとタスクバー

PCにサインインすると表示される最初の画面が「デスクトップ」です。画面下にある横長のバーを「タスクバー」と呼びます。

[スクリーンショット: Windows 11のデスクトップとタスクバー]

  • スタートボタン: アプリケーションの一覧を表示したり、PCの設定を変更したりできます。
  • 検索: PC内のファイルやアプリ、Web上の情報を検索できます。
  • アプリのアイコン: よく使うアプリをピン留めしておくと、ワンクリックで起動できて便利です。

3. Wi-Fiに接続する方法

  1. タスクバーの右側にある、ネットワークアイコン（扇形のアイコン）をクリックします。
  2. 利用可能なWi-Fiネットワークの一覧が表示されます。
  3. 接続したいネットワーク名（SSID）の横にある「接続」ボタンをクリックします。
  4. パスワード（セキュリティキー）を求められたら、入力して「次へ」をクリックします。

[スクリーンショット: Windows 11のWi-Fi接続画面]

4. ファイルとフォルダーの基本

「エクスプローラー」というアプリを使うと、PC内のファイルやフォルダーを管理できます。タスクバーの黄色いフォルダーのアイコンをクリックして起動します。

  • 新しいフォルダーの作成: 何もないところを右クリックし、「新規作成」>「フォルダー」を選択します。
  • 名前の変更: ファイルやフォルダーを右クリックし、「名前の変更」を選択します。
  • コピー＆貼り付け: ファイルを右クリックして「コピー」を選び、貼り付けたい場所で右クリックして「貼り付け」を選択します。

[スクリーンショット: Windows 11のエクスプローラー画面]

中級者向け案内 (Windows 11)

ショートカットキーを覚えて、作業をスピードアップしましょう。

基本的なショートカットキー

  • Ctrl + C: 選択した項目をコピーする
  • Ctrl + X: 選択した項目を切り取る
  • Ctrl + V: コピーまたは切り取った項目を貼り付ける
  • Ctrl + Z: 操作を元に戻す
  • Ctrl + A: すべての項目を選択する
  • Alt + Tab: 開いているアプリを切り替える
  • Windows ロゴキー + D: デスクトップを表示/非表示にする
  • Windows ロゴキー + L: PCをロックする

新しい便利なショートカットキー (Windows 11)

  • Windows ロゴキー + W: ウィジェットを表示する
  • Windows ロゴキー + A: クイック設定を開く
  • Windows ロゴキー + N: 通知センターを開く
  • Windows ロゴキー + Z: スナップレイアウトを開く

上級者向け案内 (Windows 11)

基本的なトラブルシューティングについて解説します。

使用状況の確認と基本的なトラブルシューティング

タスクマネージャーの活用:

  1. ショートカットキー Ctrl + Shift + Esc を押してタスクマネージャーを起動します。
  2. 「プロセス」タブで、現在実行中のアプリケーションやバックグラウンドプロセスを確認できます。
  3. 「パフォーマンス」タブでは、CPUやメモリ、ネットワークの使用率を確認できます。

[スクリーンショット: Windows 11のタスクマネージャー]

基本的なトラブルシューティング:

  • 再起動: まず試すべき最も簡単で効果的な方法です。
  • アプリケーションの強制終了: アプリがフリーズした場合、タスクマネージャーで対象のアプリを強制終了します。
  • Windows Updateの確認: 「設定」 > 「Windows Update」を開き、システムが最新の状態であることを確認します。

[スクリーンショット: Windows 11のWindows Update設定画面]

システム管理者向け案内 (Windows 11)

より高度なトラブルシューティングと、システム管理ツールについて解説します。

イベントビューアーの活用

Windowsは、システムやアプリケーションの重要な出来事を「イベント」として記録しています。イベントビューアーを使うことで、エラーの原因究明に役立つ詳細な情報を得ることができます。

  1. スタートボタンを右クリックし、「イベントビューアー」を選択します。
  2. 左ペインの「Windows ログ」を展開し、「システム」や「アプリケーション」のログを確認します。
  3. エラーや警告レベルのログに注目し、問題が発生した時刻と照らし合わせて原因を探ります。

[スクリーンショット: Windows 11のイベントビューアー]

回復オプションとシステムの復元

PCが正常に起動しない、または動作が著しく不安定な場合、Windowsの回復機能を利用します。

アクセス方法: 「設定」 > 「システム」 > 「回復」

  • このPCをリセットする: 個人用ファイルを保持したまま、またはすべて削除して、Windowsを再インストールします。最終手段として考えます。
  • 詳細スタートアップ: USBドライブやDVDから起動したり、システムの復元ポイントを使ったり、セーフモードで起動したりする場合に使用します。

[スクリーンショット: Windows 11の回復オプション画面]

システムの復元ポイント: 定期的に作成されるシステムの「スナップショット」です。問題が発生する前の時点にPCの状態を戻すことができますが、ユーザーファイルには影響しません。

グループポリシーとレジストリについて

生徒用のPCでは、コマンドプロンプトやレジストリエディタなどの管理ツールへのアクセスが、通常はグループポリシーによって制限されています。これらの設定は、Active Directory環境下でドメインコントローラーから一元管理されます。

個別のPC設定の確認や変更が必要な場合は、システム管理者が管理者権限でサインインし、`gpedit.msc`（ローカルグループポリシーエディター）や`regedit`（レジストリエディター）を使用します。

注意: レジストリの編集はシステムの動作に深刻な影響を与える可能性があります。必ず事前にバックアップを取るか、テスト環境で検証してください。

ネットワークトラブルの基本切り分け

「インターネットに繋がらない」という問い合わせを受けた際の、基本的な切り分け手順です。

  1. 物理接続の確認: まず、LANケーブルがPCとハブにしっかり接続されているか、抜けていないかを確認します。
  2. IPアドレスの確認:
    • コマンドプロンプトを開き、ipconfig と入力します。
    • 表示された「IPv4 アドレス」が、意図したセグメントのIPアドレス（例: 192.168.1.x）になっているか確認します。169.254.x.x のようなアドレスは、DHCPサーバーからIPアドレスを取得できていないことを示します。
  3. 疎通確認 (ping):
    • ping 8.8.8.8 を実行し、GoogleのDNSサーバーと通信できるか確認します。これで応答があれば、インターネットまでの物理的な経路は正常である可能性が高いです。
    • ping www.google.com を実行し、名前解決ができて通信できるか確認します。8.8.8.8には通るが、ドメイン名で通らない場合は、DNSの設定に問題がある可能性が考えられます。

[スクリーンショット: コマンドプロンプトでのipconfigとpingの実行結果]
