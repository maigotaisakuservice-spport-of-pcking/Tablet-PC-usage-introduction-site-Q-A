ホーム

初級者向け案内 (Windows 11)

1. 電源の入れ方と切り方

電源を入れる: PC本体の電源ボタンを押します。

電源を切る（シャットダウン）:

  1. 画面下のスタートボタン（Windowsロゴ）をクリックします。
  2. 表示されたメニューの右下にある電源アイコンをクリックします。
  3. 「シャットダウン」を選択します。
power_settings_new

2. デスクトップとタスクバー

PCにサインインすると表示される最初の画面が「デスクトップ」です。画面下にある横長のバーを「タスクバー」と呼びます。

desktop_windows

3. Wi-Fiに接続する方法

  1. タスクバーの右側にある、ネットワークアイコン（扇形のアイコン）をクリックします。
  2. 利用可能なWi-Fiネットワークの一覧が表示されます。
  3. 接続したいネットワーク名（SSID）の横にある「接続」ボタンをクリックします。
  4. パスワード（セキュリティキー）を求められたら、入力して「次へ」をクリックします。
wifi

4. ファイルとフォルダーの基本

「エクスプローラー」というアプリを使うと、PC内のファイルやフォルダーを管理できます。タスクバーの黄色いフォルダーのアイコンをクリックして起動します。

folder_open

中級者向け案内 (Windows 11)

ショートカットキーを覚えて、作業をスピードアップしましょう。

基本的なショートカットキー

  • Ctrl + C: 選択した項目をコピーする
  • Ctrl + V: 貼り付ける
  • Ctrl + Z: 元に戻す

上級者向け案内 (Windows 11)

生徒のPC利用状況の確認や、基本的なトラブルシューティングについて解説します。

使用状況の確認と基本的なトラブルシューティング

タスクマネージャーの活用:

  1. ショートカットキー Ctrl + Shift + Esc を押してタスクマネージャーを起動します。
assessment

基本的なトラブルシューティング:

  • 再起動: 多くの小さな問題は再起動で解決します。
  • Windows Updateの確認: 「設定」 > 「Windows Update」を開き、システムが最新の状態であることを確認してください。
system_update

オンライン授業ツールの活用ヒント

ビデオ会議ツール（Microsoft Teams, Zoomなど）や、オンラインホワイトボードを活用する際のヒントです。

  • マイクとカメラの確認: 授業を開始する前に、「設定」>「プライバシーとセキュリティ」>「マイク」および「カメラ」で、使用したいアプリにアクセス許可が与えられているか確認しましょう。
videocam
